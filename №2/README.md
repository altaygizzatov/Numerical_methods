
# Численное решение уравнения диффузии
## Уравнение диффузии
  $$\frac{\partial \Phi}{\partial t} = D \frac{\partial^2 \Phi}{\partial^2 x}$$
  
  $$ x \in [0, 10]$$
  
  $$ D = 1 $$
## Граничные условия
  $$\Phi(10, t) = 5$$
  
  $$\Phi(0, t) = 0$$
## Начальные условия
$$
\Phi(x, 0)=
\begin{cases}
0 & \quad x \in [0, 5)\\ 
5 & \quad x \in [5, 10)
\end{cases}
$$
